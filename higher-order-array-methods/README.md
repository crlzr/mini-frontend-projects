filter – Return only users whose city starts with the letter “G”.

reduce – Calculate the sum of all users’ IDs.

find – Find the first user whose company name includes “Group”.

some – Check if any user’s zipcode starts with “9”.

every – Check if all users have a suite that contains “Apt”.

sort – Sort users alphabetically by their company name.

forEach – Log each user’s name and email in a formatted string.

flatMap – Create a single array of all users’ street and suite combined.

includes (on arrays you create) – Check if any city in your newArray is “Gwenborough”.

splice / slice – Create a new array with only the first 5 users.
